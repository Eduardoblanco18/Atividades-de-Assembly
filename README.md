# Atividades-de-Assembly

Este repositório contém as atividades realizadas durante a disciplina **Organizações de Sistemas Computacionais** no curso de Engenharia de Computação da PUC‑Campinas.

## Objetivo

O objetivo é reunir todos os “labs” em linguagem Assembly que foram desenvolvidos, de modo que sirvam como **registro de aprendizado**, **portfólio acadêmico** e **referência futura** para você — e para qualquer pessoa que queira estudar Assembly a partir de exemplos reais.

## Estrutura do Repositório

- `Lab_01/` até `Lab_11/` — Cada pasta corresponde a uma atividade distinta da disciplina.  
- Dentro de cada pasta você encontrará os arquivos de código em Assembly, possivelmente com comentários, testes e documentação adicionais.  
- `README.md` (este arquivo) — apresenta o repositório, o contexto e como usar/navegar.

## Como Usar

1. Clone ou baixe o repositório:  
   ```bash
   git clone https://github.com/Eduardoblanco18/Atividades-de-Assembly.git
   ```
2. Navegue para a pasta que deseja averiguar.
3. Abra o arquivo .asm em seu editor de preferência para estudar o código, os comentários e a lógica implementada.
4. Use-o como base para entender a **linguagem de Assembly X86**, padrões de implementação e boas práticas de comentários.

## Dicas de estudos

- Comece pelas atividades mais iniciais (Lab_01, Lab_02) para entender a base antes de pular para as mais complexas.
- Compare seus próprios exercícios com os que foram feitos aqui para identificar diferenças, melhorias ou lacunas.
- Se você já está mais avançado, tente modificar ou expandir os exemplos aqui para criar versões “melhoradas” — isso ajuda muito no aprendizado!

## Contribuições

Este repositório tem intuito pessoal e acadêmico. Entranto, se te interessa o estudo de **Assembly X86** fique a vontade para pegar o código, estudá-lo, criticá-lo e melhorá-lo!
